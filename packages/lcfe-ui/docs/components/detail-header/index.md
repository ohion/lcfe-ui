# DetailHeader 详情标题

标题

### 何时使用

作为详情页面的标题，或其他地方标题分类


### 基本用法
作为标题
:::demo 

```vue
<template>
<div class="flex">
<lc-detail-header title="人员详情"/>
</div>
</template>


<style>

</style>
```

:::

### lc-detail-header

lc-detail-header 参数

| 参数 | 类型 | 默认 | 说明 |
| ---- | ---- | ---- | ---- |
|   title   |   string   |      |  标题    |


lc-detail-header 插槽

| 插槽名 | 说明 |
| ---- | ---- |
|    default  |   除标题之外的其他内容   |
